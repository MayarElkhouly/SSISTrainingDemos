# SSIS Training Demos
This repository contains hands-on demos and practice exercises from an SSIS (SQL Server Integration Services) training course focused on ETL (Extract, Transform, Load) development.

## Data Source
- AdventureWorks2014 database

## Topics Covered
- Data migration techniques
- Data Flow & Control Flow
- Dynamic SSIS packages
- Configuration files
- Event handling & checkpoints
- WMI & MSMQ tasks
- Maintenance plans & performance tuning
- Working with multiple data sources and destinations

## Tools Used
- SQL Server 2022
- SSIS (SSDT in Visual Studio)

## Purpose
- Practice SSIS development
- Build ETL skills
- Use as a portfolio and reference project
